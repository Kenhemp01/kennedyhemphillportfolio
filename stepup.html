StepUp

A step counter website that estimates daily step counts.

Project Description

StepUp is a web-based tool designed to estimate a user’s daily step count based on inputs such as age, weight, height, and waist circumference. Its primary goal is to offer a simple yet effective way to track physical activity—without the need for wearable devices.

The site features a clean, minimalistic design optimized for responsiveness across smartphones, tablets, and desktops. Dynamic JavaScript calculations instantly update results as users adjust their inputs, enhancing interactivity and convenience.

Beyond core functionality, StepUp prioritizes usability and user experience. Features include input validation, clear instructions, and visually engaging feedback to keep users informed and motivated.

Built with modern web standards, the project uses HTML5, CSS3, and vanilla JavaScript for structure, styling, and logic. Python was also incorporated to support backend functionality. Performance optimization and responsive design were key priorities throughout development.
